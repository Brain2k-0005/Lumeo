# Lumeo 4.0.0 — Per-Component Test Checklist

Docs dev server: **http://localhost:5287** — every line links straight to the page.

One row per component, ordered exactly like the docs sidebar so you can go top-to-bottom. For deep walkthroughs of the cross-cutting 4.0.0 **behaviour** changes (A1-A4, B1-B6, theming), see the companion **`RELEASE_TEST_4.0.0.md`**; this file is the breadth pass.

**How to use:** keep DevTools **Console open** the whole time — a clean console (no red errors / "is not a function" / NRE) is a pass criterion on every page. Tick each box as you go.

Legend: `[ ]` to test · mark `[x]` when it passes · note the page + what you saw on a fail.

4.0.0 tags you will see inline: **A1** Badge-controlled · **A2** clamp+indeterminate-ARIA · **A3** state-survival · **A4** live-DOM roving order · **B1** Icon/#348 · **B2** DataTable ItemKey · **B3** RadioGroup.Name · **B4** Gantt live-refresh · **B5** AsChild · **B6** misc parity · **C** OKLCH theming.
